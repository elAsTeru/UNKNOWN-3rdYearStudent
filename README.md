# VisualStudio 設定
動かなかった場合は設定を見てみてください、動作する場合は無視して大丈夫です。
## すべての構成
- 全般→プラットフォームを2019,v142
- 全般→C++言語標準をC++17以上
- 詳細→文字セットをマルチバイト
## デバッグ
- C/C++→コード生成→ランタイムライブラリをmtd
## リリース
- C/C++→コード生成→ランタイムライブラリをmt
